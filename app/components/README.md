# CampusChain 🎓🔗
**By Team Eonyx | Lead Developer: Anubhav Rajput**

CampusChain is a zero-friction, decentralized peer-to-peer ecosystem built strictly for the DTU campus. It integrates Algorand blockchain payments to solve high platform fees, eliminate cash meetup risks, and provide secure late-night delivery and skill-bartering networks for verified students.

### ⚙️ Local Setup Instructions
1. Clone the repository:
   `git clone https://github.com/your-username/campuschain.git`
2. Navigate to the project directory:
   `cd campuschain`
3. Install dependencies:
   `npm install`
4. Run the development server:
   `npm run dev`
5. Open `http://localhost:3000` and use any `@dtu.ac.in` email to bypass the gatekeeper.
